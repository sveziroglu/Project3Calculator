  • First, let's understand that code has standards. PEP 8 standards, to be exact. PEP 8 is the correct format in which to write code. You don't know what that correct format is so you would use the following code in your terminal to have Python tell you how your code should be written, what's missing, where to add the missing information and if your code is overall erroneous. This code will also provide a coverage report:

    • pytest --pylint --cov

  • Note: Plyint testing is referred to as "linting" your files and does show throughout every piece of code and folder, file etc. on Python.
  • Before testing or 'linting' can commence, we need to make sure we have pylint installed by using code:

    pip install pylint

Important Object Oriented Programming (OOP) Terminology

  • Factory: An object or function made to create other objects.
  • Instantiation:The creation of an object of a class. To see instantiation in action, use this conductor:

    __init__(self,value_a, value_b)

  • Object:A collection of data or functions used to act in code.
  • Class:A class or "Parent Class" is the main class in which other operations or "child classes" base their functionalities off of to perform operations.
  • Namespace:A set of names used to identify and refer to various objects.
  • Constructor:Initializes the objects of a class.
  • Fixture: A fixure is specific and known data that can be repeated for testing purposes.
  • Type Hint:Defines type of objects.
  • Typecast: Changing data from one type to another type of data.
  • Unit Test: Testing of the units in a program to ensure operations are doing what they are supposed to be doing.
  • Static Method: References class name or object without creating a new object.

    @staticmethod

    def add_number(value_a, value_b):

    """Adds number"""

  • Property:Describes functions.
  • Class Method:Functions specific to a class.
  • Facade: An object that provides a "mask" over more complex code.
  • Design Patterns: Show the interactions between classes or objects.
