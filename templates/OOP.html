  • In Python, object-oriented Programming (OOPs) is a programming paradigm that uses objects and classes in programming. It aims to implement real-world entities like inheritance, polymorphisms, encapsulation, etc. in the programming. The main concept of OOPs is to bind the data and the functions that work on that together as a single unit so that no other part of the code can access this data. (see link to this site in footer)

Definitions and Examlples of OOP Concepts:

  • Polymorphism: Using the same function for different types of objects.
    • Example:

    def get_result(self):

    return self.value_a + self.value_b

  • Inheritance: A class deriving properties from another class.
    • Example:

    from calc.calculations.calculation import Calculation

    In the above code we are inherting the properties (functions of the calculations) from the "calc" folder's "calculations" class to perform an operation in another class. Below you will see the code for the class "Calculations".

    def __init__(self,values: tuple):

    self.values = Calculation.convert_args_to_tuple_of_float(values)

     The above code gives another class the properties regarding values.

  • Encapsulation: To wrap or encapsulate data within one unit which cannot be modified.
    • Example:

    def add(value_a, value_b)

    In the above code we are defining on python how to add values a and b in the addition class.

    calc = Calculator()

    In the code above we are defining the object "calc".

    calc.add_number(4)

    In the code above we are using object calc to call a method "add_number".

  • Abstraction: This is the abstract class or the parent class wherein a set of methods are created to be used in child classes.
    • Example:

    """This is my calculation base class / Abstract Class"""

    class Calculation:

    def __init__(self,value_a, value_b):

    self.value_a = value_a

    self.value_b = value_b

    In the code above we see the parent class which defines the main method to be used throughout the calculator programming.
